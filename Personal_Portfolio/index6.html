Basic Calculator

This project is a working calculator that performs basic arithmetic operations (addition, substraction and multiplication) and solves quadratic equations.

Skills: HTML, CSS, JavaScript

Die Stimulator

This project is a die stimulator designed as an alternative to physical dice in board games
This python programme stimulates a six-sided die, using a Python module to generate a random number from 1 to 6 each time it is run.
The user can "roll" multiple dice at a time, and the programme automatically adds up the sum of the die rolls.

Skills: Python

Mathematics Quiz

This Python Programme sets the user a maths quiz with five questions and returns feedback on the number of questions the user got correct.
The programme generates different questions each time it is run, using the Python built-in random module to generate psuedo-random numbers with which to form the questions.

Skills: Python

Online Calculator

This project is a complex online calculator with a Python backend.
With Python's eval function, it can solve more complex equations that the previous calculator.
Also, it includes built-in values for mathematical constants such as pi (pi = 3.14) and acceleration due to gravity (g = 9.82), which aids calculations.

Skills: Python, HTML, CSS, JavaScript

Coin Counter

This coin counter programme computes the minimum number of coins required to provide exactly a given amount of change and lists the denominations, in Great British Pounds, of the coins required to provide this minimum make-up.
The programme will be useful in a retail context, as a basis for an automated cash dispensing system, such as a supermarket self-checkout, or as an aid to a human cashier.

Skills: Python

Number Guessing Game

This simple Python programme generates a random number between 1 and 10 using the randint function of Python's random module, and assigns it as a constant. The user then guesses the number, and Python returns feedback on whether the number is the same as the random number generated by Python.

Skills: Python
